# 🏠 Real Estate Price Analysis

## 📌 Project Overview
This project analyzes real estate prices across different cities using Python.

---

## 🎯 Objectives
- Analyze property price trends
- Compare city-wise prices
- Identify factors affecting house prices

---

## 📊 Analysis Performed
- Average price by city
- Correlation between size, rooms, and price
- Visualization using bar chart and scatter plot

---

## 📈 Key Insights
- Munich has the highest property prices
- Property size strongly affects price
- Larger houses tend to be more expensive

---

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
